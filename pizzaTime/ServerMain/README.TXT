PROJECT TITLE: Pizza Time
PURPOSE OF PROJECT: To create a service which can be used to order and create pizzas
VERSION or DATE: 1.0
HOW TO START THIS PROJECT: Run pizzaMaker class in ServerMain package 
and then run Client class in Client package
AUTHORS: Nikita Vasilyev, Daksh Guard, Shugo Shibazaki, Neil Patel 
USER INSTRUCTIONS: When the client instructions pop up, answer the questions 
about the pizza you want to order. Then on the ServerMain, create the pizza with the requested
toppings by clicking on the topping of choice and then clicking on the pizza at the desired location.
Repeat the topping placement until done. Once finished click red button.
